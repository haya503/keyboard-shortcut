# Windows画面キャプチャーショットカット

|  No   | key                  | description                          |
| :---: | :------------------- | :----------------------------------- |
|   1   | `Print Scrren`       | 画面全体をキャプチャ                 |
|   2   | `Alt`+`Print Screen` | 選択されているウインドウをキャプチャ |
|   3   | `Win`+`Shift`+`S`    | 任意のサイズでのキャプチャ           |
